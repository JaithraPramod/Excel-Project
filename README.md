# Excel-Project
Vitamin Deficiency and Associated Health Conditions

This project focuses on analyzing vitamin deficiency patterns and their association with various health conditions. The objective is to identify common deficiencies, understand their possible health impacts, and derive meaningful insights using Excel-based data analysis techniques.

# Objective
-Analyze the distribution of vitamin deficiencies
-Identify commonly associated health conditions
-Understand patterns between nutrition and health issues
-Present insights using Excel charts and pivot tables

# Tools Used
-Microsoft Excel
-Pivot Tables
-Charts (Bar, Pie, Line)
-Data Cleaning Functions
-Basic Statistical Analysis

# Dataset Description
-Age group
-Gender
-BMI category
-Diet type
-Vitamin deficiency status
-Sun  exposure 
-Diagnosed diseases

# Data Cleaning & Preparation
The following steps were performed:

-Removed duplicate entries
-Handled missing values
-Standardized categorical data
-Organized dataset for analysis
-Created structured tables for pivot analysis

# Analysis
The following analyses were conducted using Excel:

Vitamin deficiency distribution across individuals
Most common health conditions linked to deficiencies
Relationship between age groups and deficiencies

# Insights
Anaemia  is the most commonly occurring deficiency disease.
Vitamin deficiency pattern varies significantly across age groups, diet types and BMI categories.
Low bone exposure is strongly associated with higher bone pain cases, indicating vitamin D deficiency risks

# Conclusion
This analysis highlights the importance of balanced nutrition and its direct impact on health. The findings can help in early identification of deficiency-related risks and support better health awareness.
